## What is Magic Film?

Magic Film is a web platform that generates **personalized cinematographic analysis powered by artificial intelligence**. Unlike IMDb or Letterboxd — which show user-written reviews with inconsistent quality — Magic Film produces its own structured, AI-generated analysis adapted to the character of each film.

The system automatically detects whether a film is **mainstream entertainment** or **auteur cinema**, and generates a completely different analysis for each case. All analysis is delivered in Spanish, organized in thematic layers, and stored in a database to avoid redundant API calls.

## How it works

```
User types "Midsommar"
        ↓
Frontend sends search to backend
        ↓
Backend queries TMDB → retrieves genres, keywords, data
        ↓
Internal classifier → DEEP or ENTERTAINMENT
        ↓
Does analysis exist in DB?
   ├── YES → returns cached result  (fast, no cost)
   └── NO  → calls Google Gemini → saves → returns
        ↓
Frontend renders the analysis screens
```

| **Backend** | Node.js · Express.js |

| **Movie data** | TMDB API v3 |
| **Authentication** | JWT |

## User roles

Magic Film supports three distinct user roles, each with escalating permissions:

| Role | Access |
|------|--------|
| **Guest** | Search movies, view existing analyses |
| **Registered** | Everything above + generate analyses, save favorites, personalized recommendations |
| **Admin** | Full catalog management, analysis regeneration, monitoring panel |
